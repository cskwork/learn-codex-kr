## 목표

2명이 같은 코드 챌린지를 함께 푸는 코업 모드를 만든다. 룸 생성 → 초대 링크 → 공유 상태 + 룸 내 채팅 + 협력 클리어.

## Acceptance Criteria

- [ ] `/coop` 라우트 — `방 만들기` / `초대 코드로 입장` 두 액션
- [ ] WS 룸 모델: room_id, 멤버 ≤2, 진행 중인 챌린지 상태(서버 권위)
- [ ] 첫 코업 챌린지 1개 (Codex 프롬프트 협상하기) — 콘텐츠는 분리 파일
- [ ] 룸 내 채팅 (간단한 텍스트, 모더레이션 없음, 최대 200자/메시지)
- [ ] 두 명이 동시에 클리어 조건 만족 시 완료 화면 + 공유 카드
- [ ] 룸은 마지막 멤버가 나가면 5분 후 GC
- [ ] 모바일에서 채팅+퍼즐 둘 다 조작 가능

## 비목표

- 매칭/검색 (TASK-008에서)
- 음성/이모티콘
- 정교한 anti-cheat

## 운영

- 룸 메시지 broadcast는 룸 멤버에게만 — 글로벌 ws 채널과 분리
- 룸 상태는 메모리만 (재시작 시 사라짐). 영속화는 후속
